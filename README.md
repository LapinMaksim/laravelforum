# Laravel Forum
Laravel Forum Platform.

## Features
- Social Login
- Mail and Database Notification
- Algolia Instant Search
- SEO friendly URLs
- Markdown Editor
- Code Highlighter
- Tag UI
- User Profile Feed
- Email Confirmation On Registration
- Topic Views Count
- Sorting
- More...
